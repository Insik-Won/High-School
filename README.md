# High-School
고등학교 3학년 때 만든 프로그램들

### 1. Nozzle Length Ckeck
    Nozzle의 길이를 계산하는 프로그램.
    설계사인 아버지가 만들어봐라고 해서 만들게 되었다.
    Java의 swing 라이브러리로 구현했고 jre를 첨부해야해서 용량이 늘어나는 문제가 있었다. 

### 2. Calc Ckeck
    Nozzle과 Clip의 길이를 계산하는 프로그램.
    C와 c++을 배우고 MFC를 배워서 만들었다.
    아버지가 만족하셨다.

### 3. Weigh Ckeck
    여러 형강과 형상의 무게를 계산하는 프로그램.
    MFC로 만들었으며 함수 포인터 배열과 동적 배열을 구현해서 만들었다.
    클래스와 상속을 이용했다.

### 4. Bacteria Simulator
    세균이 항생제에 면역을 가지는 과정을 재현하는 프로그램
    MFC로 만들었으며 동적 UI를 구현하고 타이머와 2차원 동적 배열을 만들었다.
    생활과 과학 수업에서 수행평가용으로 만들었다.
    동적할당을 하면 어떻게 돌아가는 지 처음 깨달았다.

### 5. Koppen Climate Classification
    월별 온도와 강수량으로 지역의 기후를 쾨펜 기후 구분에 따라 구분하는 프로그램
    MFC로 만들었으며 동적 UI 전용 클래스를 만들고 드래그를 구현하고 modern c++과 STL을 사용했다.
    세계지리 수업에서 수행평가용으로 만들었다.
